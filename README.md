# motion-mmal

## Installation
1. `curl -LOk https://github.com/kevincolten/motion-mmal/archive/master.zip`
2. `unzip master.zip`
3. `cd motion-mmal-master/`
4. `chmod 777 ./motion`
5. `chmod -x ./motion`

## Run
6. `./motion -c motion-mmalcam-both.conf`
